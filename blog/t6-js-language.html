Basic Syntax - In Brackets

Parentheses () encapsulate 1) either some requisite conditions for a function to operate - while (theSky == blue) { doDaytimeStuff }, or 2) parameters to be fed into a function - var addNumbers = function (num1, num2) { return p1 + p2 } so that addNumbers(1, 2) will equal 3. They can also be used to 3 separate sections of an equation to be performed in a particular sequence, differentiating (num1 + num2) * num3 from num1 + num2 * num3 for example.

Square brackets [] are used exclusively to define arrays - storing values in a sequence, referenced by numbers, counting from zero not one. For example: var myArray = [ num1, num2, obj1, obj2 ]. To access num1, we call for the first (zero-th) item in that array, myArray[0] which === num1. For obj1, we access it with myArray[2].

Curly braces {} define an object, which is like an array, but with values accessed using keys (ie strings of text) rather than a number sequence. They can also represent the bounds of the body of a function (see above usage under Parentheses)

Single and double quote marks "" or '' are used to encapsulate strings, and are completely interchangeable, but if you open a pair of quotes, you must close the pair with the same type of quote.
